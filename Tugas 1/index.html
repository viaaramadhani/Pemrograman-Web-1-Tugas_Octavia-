HTML (Hypertext Markup Language)


About

  • Sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser
  • HMTL berupa kode-kode tag yang mengintruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan
  • HTML saat ini merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Web Consortium (W3C)

Ciri-Ciri Dokumen HTML

  • Ekstensi file berupa .htm atau .html
  • Non case sensitive
  • Terdiri atas tag-tag pembuka dan penutup
  • Tag-tag saling berpasangan & bersarang

Struktur HTML

  1. Document Information
  2. Document Header
    • Page Title
      "Judul dari halaman web"
    • Scripting
      "Tempat client -side script yang disertakan (javascript, vbsript, jsript)"
    • Style
      "Dipergunakan untuk mengatur bagaimana sebuah halaman web dengan berbagai komponennya hendak ditampilkan ke dalam browser"
    • Meta
      "Meta tags, description & keywords untuk mempermudah search engine dalam melakukan indexing"

  3. Document Body

    Bagian dari dokumen web yang akan ditampilkan ke user.
    Bagian ini memuat:

    1. Teks & gambar
    2. Link
    3. Server Side Script (PHP, ASP, JSP)
    4. Multimedia and Special Programmed Events (Shockwave, SWFs, Java Applets, online video
